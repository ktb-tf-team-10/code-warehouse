# Task
청첩장 3페이지 (장소 안내) 이미지를 생성합니다.

# Layout Structure
```
┌─────────────────────────────────┐
│   [테두리 프레임: {{border}}]    │
│                                 │
│        [예식 정보]               │
│                                 │
│   {{wedding_date}}              │
│   {{wedding_time}}              │
│                                 │
│   ─────────────                 │
│                                 │
│      [장소 정보]                 │
│                                 │
│   {{venue}}                     │
│   {{address}}                   │
│                                 │
│   ─────────────                 │
│                                 │
│   ┌─────────────────────────┐   │
│   │                         │   │
│   │     [지도 이미지]        │   │
│   │   - Google Maps         │   │
│   │   - 예식장 위치 표시     │   │

```

# Inputs
- **wedding_date**: {{wedding_date}}
- **wedding_time**: {{wedding_time}}
- **venue**: {{venue}}
- **address**: {{address}}
- **floor_hall**: {{floor_hall}} (예: "3층 그랜드홀")
- **map_image**: Google Maps Static API로 생성된 지도 이미지
- **style_image**: 배경 스타일 참조 이미지
- **border_design_id**: {{border_design_id}}

# Layout Specifications

## 예식 정보 영역 (상단 10-25%)
- **배치**: 페이지 상단, 중앙 정렬
- **여백**: 상단 60px, 좌우 40px

### 날짜 표시
- **서체**: 고딕체 (명확한 가독성)
- **크기**: 20-22pt
- **색상**: #2C3E50 (다크 그레이)
- **형식 예시**:
  ```
  2025년 4월 12일 토요일
  ```

### 시간 표시
- **서체**: 날짜와 동일
- **크기**: 18-20pt
- **색상**: #555555
- **형식 예시**:
  ```
  오후 2시 30분
  ```

## 구분선
- **스타일**: 얇고 우아한 수평선
- **길이**: 전체 너비의 30-40%
- **두께**: 1-2px
- **색상**: 테두리 디자인과 조화

## 장소 정보 영역 (중앙 30-50%)
- **배치**: 페이지 중앙 상단
- **정렬**: 중앙 정렬

### 예식장명
- **서체**: 명조체 (우아함)
- **크기**: 24-26pt (가장 크게 강조)
- **색상**: #1A1A1A (진한 블랙)
- **스타일**: 볼드
- **예시**: "더 클래식 500"

### 주소
- **서체**: 고딕체
- **크기**: 14-16pt
- **색상**: #666666
- **줄바꿈**: 자동 (너무 길면 2줄)
- **예시**:
  ```
  서울특별시 강남구 테헤란로 123
  3층 그랜드홀
  ```

### 층/홀 정보
- **서체**: 주소와 동일
- **크기**: 14pt
- **색상**: #888888
- **형식**: 별도 줄로 표시

## 지도 이미지 영역 (하단 55-90%)
- **배치**: 페이지 하단
- **크기**: 전체 너비의 80%, 높이 400-500px
- **여백**: 상단 20px, 좌우 60px, 하단 40px
- **스타일**:
  - 둥근 모서리 (border-radius: 12px)
  - 얇은 테두리 (1px solid #DDDDDD)
  - 그림자 효과 (subtle drop shadow)

### 지도 이미지 처리
- Google Maps Static API로 생성된 지도 그대로 사용
- 예식장 위치에 커스텀 마커 표시
- 충분한 확대 수준 (zoom level 15-17)
- 주변 랜드마크 포함

## 교통 정보 (선택 사항)
- **배치**: 지도 아래 또는 옆
- **서체**: 고딕체
- **크기**: 12-13pt
- **색상**: #777777
- **아이콘**: 지하철, 버스, 주차 아이콘
- **예시**:
  ```
  🚇 강남역 5번 출구 도보 3분
  🚌 간선버스 146, 740
  🅿️ 건물 내 주차 가능 (2시간 무료)
  ```

# Border Design (page1, page2와 동일)
- border1: 클래식 프레임
- border2: 플로럴 프레임
- border3: 미니멀 프레임
- border4: 로맨틱 프레임

# Background Design
- 스타일 참조 이미지 분위기 반영
- 지도 영역은 배경 없음 (순백 또는 연한 그레이)
- 상단 정보 영역에만 은은한 배경 패턴
- 배경 투명도: 15-25% (매우 연하게)

# Typography Hierarchy
1. **예식장명** (가장 크고 강조)
2. **날짜** (두 번째 강조)
3. **시간, 주소** (중간)
4. **교통 정보** (가장 작게)

# Color Palette

## 공통
- 메인 텍스트: #1A1A1A ~ #2C3E50
- 보조 텍스트: #666666 ~ #888888
- 지도 테두리: #DDDDDD
- 배경: 스타일 이미지 기반 + 흰색 기본

## 강조 요소
- 예식장명: 테두리 디자인 메인 컬러와 조화
- 아이콘: 은은한 컬러 (교통수단별 구분)

# Map Specifications

## Google Maps Static API 파라미터 예시
```
- center: [latitude],[longitude]
- zoom: 16
- size: 800x500
- markers: color:red|[latitude],[longitude]
- style: feature:poi|visibility:simplified
```

## 지도 스타일링
- 심플하고 깔끔한 맵 스타일
- 불필요한 POI 최소화
- 도로명, 랜드마크는 유지
- 커스텀 마커: 하트 또는 핀 (빨간색/골드)

# Output Specifications
- **해상도**: 1080 x 1920 픽셀 (9:16 세로)
- **포맷**: PNG
- **색상 공간**: sRGB
- **품질**: 고품질

# Accessibility Considerations
- 모든 텍스트 명확히 읽을 수 있을 것
- 지도와 배경 대비 충분히 확보
- 색각 이상자도 구분 가능한 컬러 사용
- 아이콘 + 텍스트 병행 표기

# Important Notes
- 예식장명과 주소는 정확히 표기 (오타 절대 금지)
- 날짜, 시간 정보 검증 필수
- 지도 이미지는 고해상도 유지
- 배경은 지도 가독성을 해치지 않을 것
- 전체적으로 정보 전달에 집중 (장식 최소화)
