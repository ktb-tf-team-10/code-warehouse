# Task
청첩장 1페이지 (커버) 이미지를 생성합니다.

# Layout Structure
```
┌─────────────────────────────────┐
│   [테두리 프레임: {{border}}]    │
│                                 │
│   ┌─────────────────────────┐   │
│   │                         │   │
│   │   [웨딩 사진 배치]       │   │
│   │   - 중앙 정렬           │   │
│   │   - 전체 면적의 70%     │   │
│   │                         │   │
│   └─────────────────────────┘   │
│                                 │
│   {{groom_name}} ♥ {{bride_name}}│
│   (하단 중앙, 우아한 서체)       │
│                                 │
│   [배경 디자인: {{style}}]       │
│                                 │
└─────────────────────────────────┘
```

# Inputs
- **wedding_image**: 사용자가 업로드한 웨딩 사진 (사진관 촬영본)
- **style_image**: 사용자가 선택한 배경 스타일 참조 이미지
- **border_design_id**: {{border_design_id}}
- **groom_name**: {{groom_name}}
- **bride_name**: {{bride_name}}

# Border Design Specifications

## border1 (클래식 프레임)
- 우아한 골드 테두리
- 빈티지 장식 모티브
- 전통적인 프레임 스타일
- 색상: #D4AF37 (골드), #8B7355 (브론즈)

## border2 (플로럴 프레임)
- 꽃무늬 장식 테두리
- 부드러운 꽃잎 패턴
- 로맨틱한 분위기
- 색상: #FFB6C1 (핑크), #FFFFFF (화이트), #98D8C8 (민트)

## border3 (미니멀 프레임)
- 심플한 라인 테두리
- 모던하고 깔끔한 디자인
- 여백의 미 강조
- 색상: #2C3E50 (다크 그레이), #ECF0F1 (라이트 그레이)

## border4 (로맨틱 프레임)
- 하트와 리본 장식
- 사랑스러운 디테일
- 따뜻한 감성
- 색상: #FF69B4 (핫핑크), #FFD700 (골드), #FFFFFF (화이트)

# Design Rules

## 웨딩 사진 처리
- **원본 유지**: AI 가공 없이 업로드된 사진 그대로 사용
- **배치**: 페이지 중앙, 세로 방향 기준 상단 20% ~ 하단 50% 영역
- **크기**: 전체 페이지의 70% 면적
- **여백**: 사진 주변 충분한 여백 확보 (최소 40px)

## 배경 디자인 적용
- 스타일 참조 이미지(style_image)의 색감, 패턴, 분위기 반영
- 웨딩 사진을 압도하지 않는 수준의 은은한 배경
- 배경 투명도: 30-50%

## 이름 배치
- 하단 중앙 정렬
- 웨딩 사진 아래 20-30px 여백
- 서체: 우아한 명조체 또는 세리프체
- 크기: 24-28pt
- 색상: 테두리 디자인과 조화
- 형식: "{{groom_name}} ♥ {{bride_name}}"

## 테두리 프레임 적용
- 전체 페이지 외곽 테두리
- 테두리 두께: 8-12px
- 모서리 장식: 각 border_design에 맞는 코너 디테일

# Output Specifications
- **해상도**: 1080 x 1920 픽셀 (9:16 세로)
- **포맷**: PNG (투명도 지원)
- **색상 공간**: sRGB
- **품질**: 고품질, 압축 최소화

# Important Notes
- 웨딩 사진은 절대 왜곡하지 말 것 (비율 유지 필수)
- 텍스트는 명확히 읽을 수 있어야 함
- 배경과 테두리가 조화를 이루도록
- 전체적으로 고급스럽고 우아한 느낌 유지
