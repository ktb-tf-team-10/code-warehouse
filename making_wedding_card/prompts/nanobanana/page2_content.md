# Layout Specifications

## 인사말 영역 (상단 15-40%)
- **배치**: 페이지 상단, 좌우 중앙 정렬
- **여백**: 상단 80px, 좌우 60px
- **서체**: 우아한 명조체 또는 고딕체
- **크기**: 16-18pt
- **줄간격**: 1.8-2.0
- **색상**: #2C3E50 (다크 그레이) 또는 테두리 메인 컬러
- **정렬**: 중앙 정렬 또는 양쪽 정렬

## 구분선 (Divider)
- **스타일**: 얇고 우아한 수평선
- **길이**: 전체 너비의 30-40%
- **두께**: 1-2px
- **색상**: 테두리 디자인과 조화

## 초대 문구 영역 (중앙 45-65%)
- **배치**: 페이지 중앙
- **서체**: 인사말과 동일 또는 조화로운 서체
- **크기**: 18-20pt (인사말보다 약간 크게)
- **강조**: 신랑신부 이름 볼드 처리
- **색상**: 인사말과 동일
- **정렬**: 중앙 정렬

## 부모님 성함 영역 (하단 70-90%)
- **배치**: 페이지 하단, 좌우 중앙 정렬
- **서체**: 명조체 (정중한 느낌)
- **크기**: 14-16pt
- **줄간격**: 1.5
- **형식**:
  ```
  신랑: [이름]
  [아버지명] · [어머니명]의 아들

  신부: [이름]
  [아버지명] · [어머니명]의 딸
  ```
- **색상**: #555555 (미디엄 그레이)

# Background Design
- 스타일 참조 이미지(style_image)의 분위기 반영
- 텍스트 가독성을 해치지 않는 수준의 은은한 패턴
- 배경 투명도: 20-30% (page1보다 더 연하게)
- 텍스트가 주인공

# Typography Rules

## 가독성 우선
- 명조체/고딕체 중 스타일에 맞게 선택
- 자간: -5 ~ 0 (자연스러운 간격)
- 단어 간격: 표준
- 대비비: 최소 4.5:1 (WCAG AA 기준)

## 계층 구조
1. **초대 문구** (가장 크고 강조)
2. **인사말** (중간 크기)
3. **부모님 성함** (작고 정중하게)

## 텍스트 정렬
- 인사말/초대: 중앙 정렬
- 부모님 성함: 좌측 정렬 또는 중앙 정렬
- 여백 충분히 확보

# Color Palette

## formal/classic tone
- 메인: #2C3E50 (다크 그레이)
- 포인트: #D4AF37 (골드)
- 배경: #FEFEFE (아이보리 화이트)

## romantic tone
- 메인: #8B4789 (퍼플)
- 포인트: #FF69B4 (핑크)
- 배경: #FFF5F7 (로즈 화이트)

## modern/minimal tone
- 메인: #1A1A1A (블랙)
- 포인트: #3498DB (블루)
- 배경: #FFFFFF (순백)

# Output Specifications
- **해상도**: 1080 x 1920 픽셀 (9:16 세로)
- **포맷**: PNG
- **색상 공간**: sRGB
- **품질**: 고품질

# Important Notes
- 모든 텍스트는 명확히 읽을 수 있어야 함 (최우선)
- 긴 문장은 자동 줄바꿈 (단어 단위)
- 부모님 성함은 정확히 표기
- 배경은 텍스트를 방해하지 않는 수준
- 전체적으로 우아하고 격조 있는 느낌 유지
